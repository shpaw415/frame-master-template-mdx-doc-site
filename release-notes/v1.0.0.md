## v1.0.0

### Highlights

- **Compatibility:**

  - Full support for frame-master versions 3.0.x through 3.1.x.
  - Ensures seamless integration with the latest plugin lifecycle and build orchestration.

- **Features:**

  - Converts MDX files to JavaScript for use in frame-master projects.
  - Supports both `.mdx` file extensions.

- **Improvements:**

  - Improved error handling and clearer error messages for invalid MDX syntax.
  - Optimized loader performance for faster builds.
