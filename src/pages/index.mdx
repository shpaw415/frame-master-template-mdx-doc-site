## Quick Start

```bash
# Install dependencies
bun install

# Start development server
bun run dev

# Build for production
bun run build
```

## Configuration

Customize your site by editing `src/site.config.ts`:

```typescript
export const siteConfig = {
  name: "My Documentation",
  description: "Your site description",
  // ... more options
};
```
